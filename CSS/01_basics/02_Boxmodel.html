CSS BOX MODEL

Content-box(default)

This is a content Box element

Width:300px+40px(padding)

+10px(border)=350px

Height:300px+40px(padding)

+10px(border)=350px

Border-box

This is a border Box element

Width:300px(including padding and border)

Height:200px(including padding and border)

Inline

this is paragraph withinline element.it takes only necessary width

Block-level

this is block level element.it takes full width available.
